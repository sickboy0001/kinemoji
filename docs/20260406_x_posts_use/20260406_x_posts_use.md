# 概要
## 機能
ボタンをついかすることで、気楽にTweet（XPost）可能になる。
オープンにしていい画面については、積極的にボタンを付けることで
自分の作成した情報を共有しやすくなる。
サービス使ってみようという動機づけになることも期待

## シーン
### Zst：月間WordCloud、タグのチャート
### Zst：コンテンツをAiサマライズした結果（公開ベースで）
### QPS：公開しているチャートのURL提示
### Choitameの依頼内容：依頼内容のサマリ
### Hadbit：チャートかな・・・・

## 利用時の要件
- 本文＋画像＋リンクで十分な気はする。
- 画像
  - オリジナル、WordCloudや、Chartなどインパクトあるものが最適
- リンクは誰でも参照可能なURLを提示すること
- 本文は、生成AIからの生産物がベストかも。日常向けに日々に対してのまとめつけれても面白いかも
  - 日々のまとめてを塚するのもありかな。
- 横のメニューはゲスト操作可能なものへのつながりがベスト
  

# 基本仕様
## X（旧Twitter）が公式に用意している「Web Intent（インテント）」機能
https://x.com/intent/tweet
クエリパラメータ（text / url など）

## ログインとの関係
Xにログイン済み → すぐ投稿画面
未ログイン → ログイン画面 → その後投稿画面
